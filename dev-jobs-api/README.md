# Dev Jobs API

A REST API for the Dev Jobs Board app.

## Tech Stack
Node.js, Express

## Getting Started

git clone https://github.com/BillZevg/dev-jobs-api.git
cd dev-jobs-api
npm install
node server.js

## API Endpoints
| Method | Endpoint        | Description        |
|--------|-----------------|--------------------|
| GET    | /api/jobs       | Get all jobs       |
| GET    | /api/jobs/:id   | Get one job        |
| POST   | /api/jobs       | Create a job       |
| DELETE | /api/jobs/:id   | Delete a job       |
