# Dev Jobs Board

A React app to search and browse developer job listings.

## Tech Stack
React, React Router, Vite

## Features
- Search jobs by title, company or technology
- Filter results in real time
- Click any job to view full details
- Connected to a live REST API

## Getting Started
git clone https://github.com/BillZevg/dev-jobs-board.git
cd dev-jobs-board
npm install
npm run dev

## API
This project connects to the Dev Jobs API:
https://github.com/BillZevg/dev-jobs-api
